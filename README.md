# FortressForesight

A home protection app that helps homeowners prepare for, document, and recover from property disasters.

## What It Does

1. **Property Profile** - Enter an address, get flood zone info, wildfire risk, storm risk, and a simple risk scorecard
2. **Document Storage** - Upload and organize photos of insurance policies, receipts, inspections, and warranties
3. **Readiness Center** - Premade and custom checklists for buying a home, seasonal maintenance, and post-storm prep
4. **Recovery Tracker** - Document damage room by room, log expenses, track claim status and next steps
5. **Knowledge Base** - Plain-language explanations of deductibles, flood insurance, public adjusters, and more

## Tech Stack

- React (PWA - installable to home screen)
- Firebase (auth, Firestore database, cloud storage)
- FEMA flood map API
- NOAA weather/storm data
- Google Maps / geocoding

## Getting Started
